# ci-info-action

`ci-info-action` is a GitHub Action to retrieve data related to the current workflow run.
This action does **not** depend on [ci-info](https://github.com/suzuki-shunsuke/ci-info).

## Example

All inputs are optional.

```yaml
- uses: suzuki-shunsuke/ci-info-action@latest
```
